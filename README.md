## Project Description

A node.js application that generates a professional README. This application removes the mundane and repetitive task of creating your general outline of your README file.

## Installation

1. Clone code from repository


2. Open terminal inside readme-be-done and install the required npm packages `npm i fs && inquirer`

## Usage

1. Clone code from repository


2. Open terminal inside readme-be-done and install the required npm packages `npm i fs && inquirer`


3. Open a new terminal inside the utils directory and type `node index.js`


4. Answer the prompts accordingly


5. Review your new README.md file and add any markdown or additonal indepth information
